## 🚀 Getting Started

### Installation

No build tools needed. It's a single HTML file.

```bash
# Clone or download
git clone https://github.com/yourname/pixelforge-ai.git

# Open directly in browser
open pixelforge-ai.html

# Or serve locally
npx serve .
python -m http.server 8080
```

### Quick Start Flow

```
  1. Open pixelforge-ai.html in any modern browser
         │
         ▼
  2. Click "GENERATE PIXEL ART" or scroll to Upload
         │
         ▼
  3. Drag & drop your image (PNG / JPG / WEBP / GIF)
         │
         ▼
  4. Studio opens automatically ──► adjust controls:
         │   ├── Pixel Size slider
         │   ├── Color Palette
         │   ├── Retro Filter
         │   └── AI Style (optional)
         ▼
  5. Hit ⬡ GENERATE PIXEL ART ⬡
         │
         ▼
  6. Zoom, inspect, edit pixels manually if needed
         │
         ▼
  7. Export as PNG / SVG / PDF  ✓
```

```
  Colors: #00F5FF (cyan)  #FF00FF (magenta)  #39FF14 (green)

```

## 🧠 Pixel Generation Algorithm

```
INPUT: Source Image (any resolution)
           │
           ▼
   ┌───────────────┐
   │  DOWNSCALE    │  Max dimension: 512px (aspect-ratio safe)
   └───────┬───────┘
           │
           ▼
   ┌───────────────┐
   │ AI PRE-FILTER │  Optional: contrast / saturation / tone shifts
   └───────┬───────┘
           │
           ▼
   ┌───────────────┐
   │  SAMPLE GRID  │  Sample one pixel per block (pixelSize × pixelSize)
   └───────┬───────┘
           │
           ▼
   ┌───────────────┐
   │ PALETTE MAP   │  For each sample → find closest RGB in palette
   │               │  Algorithm: Euclidean distance in RGB space
   │               │  Σ (R₁-R₂)² + (G₁-G₂)² + (B₁-B₂)²
   └───────┬───────┘
           │
           ▼
   ┌───────────────┐
   │  RENDER GRID  │  fillRect() per pixel block on <canvas>
   └───────┬───────┘
           │
           ▼
   ┌───────────────┐
   │ APPLY FILTER  │  Scanlines / CRT / Dither / Outline / Shadow
   └───────┬───────┘
           │
           ▼
OUTPUT: Pixel Art Canvas  ──►  PNG / SVG / PDF
```

## ✏️ Pixel Editor Mode

```
  1. Generate your pixel art
  2. Click [ EDITOR MODE ] button
  3. Pick a color from the palette swatches or color picker
  4. Click any pixel on the canvas to repaint it
  5. Toggle GRID OVERLAY for precision editing
  6. Export when done

  Cursor changes to ╋ cell mode in editor
  Each click updates the in-memory pixelData[][] grid
  Changes are preserved on export
```

## 📐 Performance Notes

- Canvas rendering optimized with `fillRect()` batch operations
- Image downscaling capped at 512px max dimension before pixel processing
- Particle system uses `requestAnimationFrame` for 60fps rendering
- `image-rendering: pixelated` CSS ensures crisp pixel display at any zoom
- Inertia scroll uses easing interpolation (not `scroll-behavior: smooth`) for cinematic feel
